# Pelozen Caller — בוט חיוג קולי בעברית בקול המשתמש

מערכת חיוג יוצא שמנהלת שיחות אנושיות בעברית **בקול המשובט של בעל המערכת**, מבררת עניין מול לידים שהסכימו לשיחה חוזרת, ומתעדת תוצאה מובנית לכל ליד.

המשתמש מפעיל יומית: בוחר תחום (מתוך ~20), טוען מנת לידים, מריץ — והבוט מחייג **ברצף** (קו אחד) דרך כרטיס סים מקומי ב-GSM gateway.

> ⚠️ **דגל משפטי — קרא לפני הרצה.**
> בישראל חיוג אוטומטי לפרסום כפוף ל**סעיף 30א** (חוק התקשורת), ל**חוק הגנת הפרטיות (תיקון 13, בתוקף 08/2025)** ולמאגר **"אל תתקשרו אליי"** (קנס עד **₪46,080 לשיחה** למספר רשום). רשות הפרטיות מצפה ל**גילוי שמדובר במערכת AI אוטומטית**.
> ההגנה היחידה במערכת זו: **הלידים הסכימו לשיחה חוזרת, וההסכמה מתועדת לכל ליד** (`consent_records`).
> בעל המערכת בחר **לא להפעיל** את הכרזת ה-AI. המתג בנוי וניתן להדלקה בקליק (`shared/compliance/config.ts`). **מומלץ אישור עו"ד פרטיות/הגנת הצרכן לפני הרצה בנפח.** מסמך זה אינו ייעוץ משפטי.

## ארכיטקטורה (DIY רזה, self-hosted)

```
Operator Panel (Next.js/Vercel)
   │ HTTP + Supabase Realtime
   ▼
Orchestrator (VPS)  ──►  Voice Pipeline (Pipecat)
   │ gates ציות           STT Deepgram he → LLM → TTS Cartesia (קול משובט)
   │ לולאת רצף             │ SIP
   ▼                      ▼
Supabase (Postgres)   Asterisk/FreeSWITCH ──SIP──► GSM gateway + סים ──► ליד
[תוצאה + תמליל טקסט]
```

## מבנה המונורפו

| תיקייה | תוכן |
|---|---|
| `shared/` | טיפוסים + קונפיג ציות, משותף ל-web ו-orchestrator |
| `orchestrator/` | לולאת הרצף, gates ציות, קמפיינים, promptCompiler, משיכת לידים, telephony adapter |
| `pipeline/` | ה-pipeline הקולי ב-Python (Pipecat): STT→LLM→TTS, barge-in |
| `web/` | פאנל התפעול (Next.js App Router, RTL) |
| `supabase/` | migrations של מודל הנתונים |

## עלות צפויה (קו אחד, רצף)
~$120–200/חודש — טלפוניה (סים בתעריף שטוח) ~$11, Deepgram ~$20, LLM ~$40–60, Cartesia ~$20–40, תשתית ~$15–45. (+ ~$120 חד-פעמי ל-GSM gateway.)
